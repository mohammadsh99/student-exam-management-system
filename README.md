# Student Exam Management System

A web application built with ASP.NET Core (.NET 6) for managing students, exams, and results.

## Features
- Manage students and their information
- Create and manage exams
- Track student results
- Responsive UI using Bootstrap

## Technologies Used
- C#
- ASP.NET Core (.NET 6)
- Razor Pages
- Entity Framework Core
- SQL Server
- Bootstrap

## Project Structure
- Pages – Razor Pages for the UI
- wwwroot – Static files (CSS, JavaScript, libraries)
- Program.cs – Application entry point
- appsettings.json – Configuration settings
